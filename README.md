# Security Monitoring Dashboard

# Project Description

A Python-based Security Monitoring Dashboard designed to collect, analyse and visualise security events from operating system logs. The project demonstrates concepts used within Security Operations Centres (SOCs), which includes, event analysis, alert generation, log ingestion and dashboard visualisation.

# Features

□ Log ingestion

□ Authentication log parsing

□ Event storage

□ Alert generation

□ Email alerts

□ Docker deployment

□ GeoIP lookup

□ Live dashboard
